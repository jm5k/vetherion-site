# Vetherion

Official landing site for the **Vetherion universe**.

- Live site: https://vetherion.com
- Play **Hexryth**: https://hexryth.vetherion.com

## What this repo is
A simple, static website (HTML/CSS/JS) used as the public hub for announcements, links, and future releases in the Vetherion universe.

## Deployment
Deployed via **Cloudflare Pages** from the `main` branch.

## Editing
Update the static files (typically `index.html`, `styles.css`, and any assets).  
Pushing to `main` triggers an automatic deploy in Cloudflare Pages.
